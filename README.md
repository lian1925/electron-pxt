# react-pxt

提供简单的 electron 工程，实现将 microbit 应用打包成桌面应用

注： This project is just a gift for friend LiChao.

## ready

1、克隆本项目

2、打包最新的 microbit 文件

3、将打包文件拷贝入 dist 文件夹(拷贝之前清空原文件。原文件为旧版本的 microbit 文件，在这里用于演示)

## Build

```bash
npm i
npm run build
```

稍等片刻，在工程目录生成打包成功的桌面应用
